# Limestone Weathering Study Data and Code

## Overview
This repository contains all datasets, scripts, and figures used in the study:

**Title:** Paleo-weathering processes and regolith development in tropical limestone terrain: Implications for resource evaluation and subsurface mapping

**Purpose:** To provide geophysical and geochemical data, along with analysis scripts, for mapping subsurface weathering architecture and evaluating limestone quality in Onigbedu, Southwestern Nigeria.

---

## Repository Contents

- `data/` – Geochemical datasets (XRF and AAS) and 2D ERI profiles
- `code/` – Python/Matlab scripts for MCIA, Mobility Ratio calculations, and data processing
- `figures/` – Figures for subsurface weathering architecture
- `LICENSE` – License files for data and code
